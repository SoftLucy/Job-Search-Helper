# Job Search Helper

A lightweight Chrome/Edge extension that adds hide buttons to job listings on Indeed, Stepstone, LinkedIn, and Arbeitsagentur. It helps you manage and temporarily hide unwanted job postings while browsing.

## Features

- Injects hide controls into supported job listing pages
- Stores hidden job listings using browser storage
- Provides a popup to view hidden count
- Temporarily show all hidden jobs
- Re-hide listings after review
- Clear storage to reset hidden job data

## Supported sites

- Indeed
- Stepstone (.com, .de, .fr, .nl)
- LinkedIn
- Arbeitsagentur
- jobboerse.arbeitsagentur.de

## Installation

1. Open your browser's extensions page.
2. Enable developer mode.
3. Load the extension as an unpacked extension from this repository folder.
